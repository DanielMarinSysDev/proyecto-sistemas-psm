Guía de Uso del Sistema

Manual interactivo y de inducción rápida para entender la automatización de carpetas, flujos y permisos de TaskCore.

Estructura del Servidor de Archivos

Al confirmar o crear una orden, el sistema genera automáticamente una estructura jerárquica en el servidor de almacenamiento. Esto evita archivos sueltos y asegura el orden del taller.

// Ruta generada físicamente:

📦 TaskCore_Archivos /

└─ 📂 Pedidos / [Año] /

└─ 📂 [Cliente_ID] - [Nombre_Cliente] /

└─ 📂 [Pedido_ID] - [Nombre_Proyecto] /

├─ 📂 Editable / (Diagnósticos / Configuración)

├─ 📂 Muestras / (Evidencias Físicas / Fotos de Falla)

└─ 📂 Salida_Impresion / (Informes / Entregables final)

¿Cómo abrir la carpeta local?

El sistema utiliza el protocolo registrado taskcore://.

En el Dashboard de Kanban o en el panel de detalles, al hacer clic en el botón de la carpeta, el navegador se comunicará con tu sistema operativo para abrir directamente el Explorador de Archivos de Windows en la carpeta del pedido correspondiente.

El flujo de Hot Folders (Watcher)

El sistema monitorea activamente las subcarpetas del servidor. Cuando el técnico deposita un archivo listo en Salida_Impresion, el monitor detecta el cambio e interactúa de la siguiente forma:

1. Creación del Enlace de Soporte

El watcher determina la estación/taller de reparación según el tipo de servicio e instala un enlace duro (hard link) en el directorio local de la estación:

  • Soporte Técnico General → D:\Soporte_Tecnico
  • Diagnósticos → D:\Diagnosticos
  • Laboratorios Hardware → D:\Laboratorio_Hardware

2. Cierre Automático

Una vez que el técnico de reparación termina el servicio y mueve el archivo/informe a la carpeta de completados de la estación, el watcher detecta el movimiento y de inmediato:

  • Mueve el Kanban a Listo para Entregar.
  • Envía notificación de finalización del servicio.

Matriz de Control de Acceso (RBAC)

Cada miembro del equipo tiene asignado un rol específico en el sistema. El sistema aplica restricciones estrictas para proteger la confidencialidad de la facturación y optimizar las tareas de cada quien.
